# Issue Management Platform

## Tech Stack

- Backend: NestJS, Drizzle ORM, PostgreSQL (Neon)
- Frontend: Next.js, TailwindCSS
- AI: Gemini API (with fallback)

## Features

- Create & manage issues
- Add discussions
- AI-based issue analysis
- Clean REST API

## Setup

### Backend

cd backend
npm install
npm run start:dev

### Frontend

cd frontend
npm install
npm run dev
